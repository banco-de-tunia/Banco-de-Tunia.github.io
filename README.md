# Banco Central de Tunía

Este es un sitio web que simula la presentación de información económica y financiera relacionada con el Banco Central de Tunía. El sitio proporciona secciones clave como el tipo de cambio, las estadísticas económicas y el mercado de valores, todo con un diseño moderno y responsivo.

## Características

1. **Barra de navegación:** Permite al usuario navegar entre las secciones de Inicio, Tipos de Cambio, Estadísticas, y Mercado de Valores.
2. **Diseño moderno y responsivo:** El sitio utiliza un degradado de colores futurista para el fondo y tiene un estilo limpio y moderno.
3. **Sección de Tipos de Cambio:** Muestra las tasas actuales de cambio para USD y EUR en relación con la moneda local, el "Tunio".
4. **Estadísticas Económicas:** Presenta estadísticas clave como la tasa de interés y el riesgo país.
5. **Mercado de Valores:** Muestra información sobre el precio y la cantidad disponible de diferentes acciones en el mercado.

## Estructura del Proyecto

El proyecto se compone de los siguientes archivos y elementos principales:

### Archivos HTML

- **index.html:** Página principal con la estructura de las secciones y los estilos aplicados.
- **Estilos:** El archivo de estilos está incluido dentro del archivo HTML como un bloque `<style>`.

### Secciones

1. **Inicio:**
   - Presenta la misión, visión y objetivos principales del Banco Central de Tunía.
2. **Tipos de Cambio:**
   - Muestra las tasas de cambio actuales para USD y EUR.
3. **Estadísticas:**
   - Presenta estadísticas clave como la tasa de interés y el riesgo país.
4. **Mercado de Valores:**
   - Muestra información sobre las acciones disponibles en el mercado, incluyendo precio y cantidad disponible.

### Diseño Responsivo

El diseño de la página es completamente responsivo, lo que significa que se adapta a diferentes tamaños de pantalla, desde escritorios hasta dispositivos móviles. Usando media queries, el diseño se ajusta para ofrecer una experiencia de usuario fluida en todas las plataformas.

## Cómo Ejecutar el Proyecto

Este proyecto es completamente estático y no requiere ningún servidor o backend para ejecutarse. Solo sigue estos pasos para verlo en funcionamiento:

1. Descarga o clona el repositorio.
2. Abre el archivo `index.html` en tu navegador favorito.
3. ¡Listo! El sitio se cargará y podrás navegar por las diferentes secciones.

## Personalización

Para personalizar el contenido o el diseño de este sitio web, puedes hacer lo siguiente:

1. **Modificar las secciones:** Cambia los textos de la misión, visión, objetivos y cualquier otro contenido que desees.
2. **Actualizar los tipos de cambio y estadísticas:** Ve a las secciones correspondientes en el HTML y actualiza las tasas de cambio y las estadísticas económicas como la tasa de interés o el riesgo país.
3. **Cambiar los estilos:** Si deseas modificar el diseño visual, puedes editar los estilos en el bloque `<style>` dentro del archivo `index.html`.

## Tecnologías Utilizadas

- **HTML5**: Estructura del contenido y la semántica de la página.
- **CSS3**: Diseño visual y responsivo utilizando propiedades modernas de CSS como `flexbox`, `box-shadow`, y `media queries`.
- **JavaScript**: Funcionalidad para la navegación entre secciones mediante el uso de la función `showSection`.
